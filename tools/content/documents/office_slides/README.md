# base_tool

Tool for base tool

## Category

General

## Parameters

No parameters.

## Returns

Returns a dictionary with:
- `success` (bool): Whether the operation succeeded
- `result` (dict): Tool-specific results
- `metadata` (dict): Additional information about the operation

## Usage Example

```python
from tools.content.documents.office_slides import Unknown

# Initialize the tool
tool = Unknown(
)

# Run the tool
result = tool.run()

# Check result
if result["success"]:
    print(result["result"])
else:
    print(f"Error: {result.get('error')}")
```

## Testing

Run tests with:
```bash
python base_tool.py  # Run standalone test
```
