## Most Common Tool Patterns

**Pattern 1: Search Tools**
- web_search, scholar_search, image_search, video_search, product_search, google_product_search
- Pattern: `{query: "search terms"}` → Returns array of results

**Pattern 2: Content Retrieval**
- crawler, summarize_large_document, onedrive_file_read, notion_read
- Pattern: `{url/id: "...", question: "..."}` → Returns content/answer

**Pattern 3: Media Generation**
- image_generation, video_generation, audio_generation
- Pattern: `{model: "...", query: "...", params: {...}}` → Returns media URL

**Pattern 4: Media Analysis**
- understand_images, understand_video, audio_transcribe, analyze_media_content
- Pattern: `{media_urls: [...], instruction: "..."}` → Returns analysis

**Pattern 5: Chart Generation**
- All mcp_c33e5f11_generate_* tools (15 tools)
- Pattern: `{data: [...], title: "...", dimensions: {...}}` → Returns chart

**Pattern 6: File Operations**
- aidrive_tool, Read, Write, MultiEdit
- Pattern: `{action/path: "...", content/edits: {...}}` → Returns status

**Pattern 7: Communication**
- gmail_search, gmail_read, email_draft, google_calendar_*
- Pattern: `{query/id: "...", params: {...}}` → Returns data/content
